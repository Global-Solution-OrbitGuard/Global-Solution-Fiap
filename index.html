Monitoramento inteligente de desastres naturais para salvar vidas

O OrbitGuard utiliza dados de satélites e sensores IoT para antecipar riscos, enviando alertas e contribuindo para prevenção de desastres.

O Problema

Os desastres naturais estão cada vez mais frequentes e imprevisíveis

Queimadas

Focos de calor se espalham rapidamente e são difíceis de estabilizar.

Secas e estiagem

Impactam a produção agrícola e o abastecimento de água para a população.

Chuvas e enchentes

Causam destruição, desabrigam famílias e colocam vidas em risco.

Umidade e deslizamentos

Áreas com solo muito úmido podem ameaçar comunidades vulneráveis.

Tecnologias Utilizadas

O OrbitGuard combina tecnologias avançadas para coletar e processar dados

Dados de satélite

Imagens via satélite NASA capazes de detectar anomalias ambientais.

Sensores IoT

Sensores que monitoram o clima e o ambiente em tempo real.

Plataforma Web

Interface intuitiva para visualização de alertas e condições climáticas e ambientais, utilizando HTML, CSS e Javascript.

Python

Analisa e processa os dados para gerar previsões de risco.

Arduino

Interação com os sensores para coleta de dados no campo.

Objetivo

Nosso objetivo é transformar dados em ações que protejam vidas e o meio ambiente

Antecipar riscos

Identificar e monitorar desastres naturais com antecedência.

Apoiar agricultores

Fornecer informações precisas para proteger lavouras.

Facilitar decisões

Oferecer dados claros e confiáveis para quem precisa agir.

Público-alvo

O OrbitGuard foi desenvolvido para atender qualquer grupo que precisa de informações

Agricultores

Planejamento agrícola com dados do clima e risco de seca.
